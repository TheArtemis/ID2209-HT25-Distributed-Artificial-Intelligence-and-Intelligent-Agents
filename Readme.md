# Setup
Use the link.sh script to make a Symlink from your repo to the the actual Gama Workspace.

```bash
# Example
./scripts/link.sh /home/username/kth/ID2209/Gama_Workspace
```

# Start
After setting up the .env file with the path to your GAMA executable, you can run the GAMA platform using the gama.sh script.

```bash
./gama.sh
```
